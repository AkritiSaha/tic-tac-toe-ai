# 🎮 Tic-Tac-Toe AI (Unbeatable)

## 🚀 Project Overview

This project is a **Tic-Tac-Toe game built using Python**, where a human player competes against an AI opponent.

The AI is designed using the **Minimax Algorithm**, a recursive decision-making algorithm from game theory that ensures the AI always plays the optimal move.

👉 Result: The AI is **unbeatable** — the best a human player can achieve is a draw.

---

## 🧠 How the AI Works (Minimax Explained)

The Minimax algorithm simulates all possible game states and chooses the best move based on outcomes.

### 🔍 Logic:

* If AI wins → score = +1
* If Human wins → score = -1
* If Draw → score = 0

The algorithm:

1. Explores all possible moves recursively
2. Assumes:

   * AI tries to **maximize** score
   * Human tries to **minimize** score
3. Chooses the move with the best outcome

👉 This guarantees optimal play every time.
