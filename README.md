# 🎮 Tic-Tac-Toe AI (Unbeatable)

## 🚀 Overview

This project is a Tic-Tac-Toe game where a human player competes against an AI opponent. The AI uses the **Minimax Algorithm**, making it unbeatable.

---

## 🧠 Features

* Human vs AI gameplay
* Unbeatable AI using Minimax
* Win / Draw detection
* CLI and GUI versions
* Interactive interface using Tkinter

---

## 🛠️ Tech Stack

* Python
* Tkinter (GUI)
* Minimax Algorithm

---



## 💡 Learnings

* Game Theory concepts
* Recursive algorithms (Minimax)
* AI decision making
* GUI development using Tkinter
